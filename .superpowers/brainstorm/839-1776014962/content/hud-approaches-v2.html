HUD Design: Approach A vs B

Click to select which direction feels right

A

Floating Top, Grounded Bottom

Semi-transparent top bar overlays the game. Solid bottom bar with clear panel borders. Spawn toggles float on the right side of the game area.

Pros

  • Top bar blends into game
  • More game area visible
  • Lighter visual weight
  • Spawn toggles always visible, separate from HUD

Cons

  • Mixed visual language (transparent + solid)
  • Floating elements may feel disconnected
B

Unified Frame

Top and bottom bars share the same solid panel treatment. Spawn toggles integrated into top bar. Consistent "cockpit" feel around the game area.

Pros

  • Consistent frame around game
  • Clear HUD boundaries
  • Traditional RTS feel
  • Everything in one visual system

Cons

  • Takes more screen space
  • Heavier visual presence
