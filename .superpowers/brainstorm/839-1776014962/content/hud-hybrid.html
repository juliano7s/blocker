Design decisions

  • Top bar: Solid frame — player info, game time, population, menu button
  • Spawn toggles: Float at top-right of game area, hotkeys 1 2 3. Dimmed = paused
  • Minimap: Clear border, labeled, viewport indicator
  • Selection: Clickable unit icons (click = select one, shift+click = deselect)
  • Commands: Icon grid with hotkey in corner, tooltip on hover
